Cara Order

Pertama yang harus diperhatikan

- Siapkan file anda -
- Pastikan file sudah siap cetak dan High Resolusi-
- Jelaskan dengan detail pesanan anda agar tidak terjadi kesalahan dalam proses pencetakan. -

Process

Offline Order

Datang Ke Store

Silahkan kunjungi store IKAPrint terdekat di wilayah Anda, lalu Costumer service kami siap melayani Anda

Mengambil Nomor Antrian

Jika sudah berada di store IKAPrint, segera ambil nomor antrian di meja Admin dan tunggu sampai nomor Anda dipanggil, selagi menunggu Anda dapat melihat-lihat katalog bahan yang ingin digunakan

Proses File

Jika nomor antrian sudah dipanggil, silakan menuju operator yang memanggil, lalu berikan file yang ingin di cetak dan berikan spesifikasi (bahan, quantity, dan finishing) dengan jelas

Estimasi Biaya

Kami akan memberikan estimasi biaya total pemesanan Anda secara langsung.

Proses Pencetakan

Setelah estimasi biaya sudah anda konfirmasi, kami akan melanjukan ke tahap cetak. Jika Quantity cetakan Anda banyak, kami akan approve 1 lembar terlebih dahulu untuk meminimalisir kesalahan

Pembayaran

Setelah proses cetak selesai, Anda bisa melakukan pembayaran di kasir, dapat dengan Cash, Transfer antar bank, dan Qris. Setlah melakukan pembayaran Jangan Lupa untuk memberikan Rating ⭐5 pada Google Maps jika Anda Puas dengan kinerja kami.

Mengirim File

Kirimkan email dan attach file yang ingin dicetak oleh kami, lalu jangan lupa menuliskan spesifikasi file yang ingin dicetak, perlu diketahui (File yang dikirim harus format PDF atau JPG, harus disertai keterangan yang jelas[bahan, ukuran, quantity, finishing], pembayaran minimal 50%dari total biaya order)

Konfirmasi File

Anda juga bisa mengirim file dengan WhatsApp kami dengan format file PDF, kami akan melakukan konfirmasi kembali untuk menghindari kesalahan dan agar file yang dicetak sesuai keinginan Anda.

Proses File

Kami akan menyiapkan file anda sesuai dengan spesifikasi file yang telah Anda berikan.

Estimasi Biaya

Kami akan mengirimkan estimasi biaya total pemesanan anda melalui WhatsApp untuk dikonfirmasi.

Proses Pencetakan

Setelah estimasi biaya sudah anda konfirmasi, kami akan melanjukan ke tahap cetak. Jika Quantity cetakan Anda banyak, kami akan approve 1 lembar terlebih dahulu untuk meminimalisir kesalahan

Pembayaran

Setelah estimasi dikirimkan dan total biaya sudah sesuai dengan pesanan Anda, segera kirimkan bukti pembayaran agar kami langsung memulai ke tahap proses cetak

Proses Pencetakan

Setelah anda mengirimkan bukti pembayaran kami akan langsung memasukan Anda kedalam antrian memulai proses cetak. Jika proses pencetakan sudah selesai kami akan memberi kabar ke Anda melalui WhatsApp

Pengambilan (Pick Up)

Jika cetakan anda telah selesai, anda dapat mengambil cetakan langsung ke store IKAPrint atau memesan jasa kurir. Jika anda memesan jasa kurir, jangan lupa untuk konfirmasi kepada admin dan mengirimkan profile kurir yang akan mengambil

Konfirmasi Barang

Segera konfirmasi ke kami apabila barang sesuai dengan Spesifikasi yang anda pesan. Apabila ada kekurangan / kesalahan produksi segera hubungi kami via WhatsApp untuk pengecekan. Jangan Lupa untuk memberikan Rating ⭐5 pada Google Maps jika Anda Puas dengan kinerja kami.
